# GestEffect Images & Screenshots

This folder contains screenshots and promotional images for the GestEffect project.

## Required Images

### `gesteffect-demo.png`
- **Size**: Screenshots of the live GestEffect interface
- **Purpose**: Featured in the main README.md under "Try It Live!" section
- **Location**: Should be placed at `docs/gesteffect-demo.png`
- **Content**: Live demo showing:
  - The neon hand tracking interface
  - Theme selector buttons (bottom-right)
  - Stats panels (FPS, hand count, gesture, spread)
  - The glowing hand wireframe on dark background
  - Record/Pause/Mute/Kill buttons

## Hugging Face Integration

The live demo is available at: **https://sam-max1-gesteffect.hf.space/**

Screenshots from the live demo can be directly referenced in documentation.
