# Contributing to GestEffect

Thank you for your interest in contributing to GestEffect! We welcome contributions from everyone.

## How to Contribute

### Reporting Bugs
- Use the GitHub Issues page
- Describe the bug clearly
- Include steps to reproduce
- Mention your Python version and OS

### Suggesting Enhancements
- Open a GitHub Issue with "Enhancement" label
- Describe the desired feature
- Explain the use case
- Provide examples if possible

### Submitting Pull Requests

1. Fork the repository
2. Create a branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Follow the code style (PEP 8 for Python)
5. Add tests if applicable
6. Commit with clear messages: `git commit -am 'Add feature: description'`
7. Push to your fork: `git push origin feature/your-feature-name`
8. Open a Pull Request

### Code Style
- Python: Follow PEP 8 conventions
- Comments: Use clear, descriptive comments
- Docstrings: Document all classes and functions
- Type hints: Include where possible

### Testing
- Test your changes locally
- Run `python -m py_compile app.py` to verify syntax
- Check that the application starts without errors
- Verify all features work as expected

### Documentation
- Update README.md if adding features
- Add code comments for complex logic
- Update QUICK_START.md if changing setup process

## Code of Conduct

### Our Pledge
We are committed to providing a welcoming and inspiring community for all. We expect all contributors to be respectful and constructive.

### Expected Behavior
- Be respectful and inclusive
- Use welcoming and inclusive language
- Be patient and understanding
- Focus on what is best for the community
- Show empathy towards others

### Unacceptable Behavior
- Harassment or discrimination
- Offensive comments or trolling
- Personal attacks
- Publishing private information
- Any conduct violating local laws

### Reporting Issues
Report inappropriate behavior to the maintainers privately.

## Questions?
- Check existing issues and documentation first
- Open a GitHub Discussion for questions
- Be specific and provide context

Thank you for contributing! 🎉
